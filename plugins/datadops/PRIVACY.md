# DatadOps Privacy Policy

DatadOps is a Codex plugin package that provides reusable workflows on top of the Datadog MCP server.

## What DatadOps stores

This plugin package does not operate its own backend service and does not independently collect, store, or transmit user data outside the MCP calls initiated by the Codex client you use.

## Third-party services

When you use DatadOps, Codex may call the Datadog MCP server to retrieve observability data required by the plugin's skills. Those requests are governed by:

- your Codex client configuration
- your Datadog account permissions and policies
- Datadog's own privacy and service terms

## Repository metadata

If you access this plugin through a public repository, standard repository hosting metadata may be processed by the hosting provider, such as GitHub account, access logs, and issue content.

## Your responsibility

You are responsible for reviewing the privacy, retention, and compliance implications of the Datadog environment and Codex environment where you run this plugin before using it with production data.

## Contact

For questions about this plugin package, use the contact details published in `.codex-plugin/plugin.json` and the linked repository.
